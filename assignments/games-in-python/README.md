
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build a playable Hangman game using Python with string handling, loops, conditional logic, and user input.

## 📝 Tasks

### 🛠️ Game implementation

#### Description

Create a text-based Hangman game where the player guesses letters to uncover a hidden word. The game should continue until the word is fully guessed or the player runs out of allowed attempts.

#### Requirements
Completed program should:

- Randomly select words from a predefined list
- Accept letter guesses and show current progress in `_ _ _` format
- Track incorrect guesses and remaining attempts
- End with a win message when the word is guessed
- End with a lose message when attempts are exhausted
- Show guessed letters and prevent duplicate guesses

### 🛠️ Optional enhancements

#### Description

Add extra polish to make the game more user-friendly and robust.

#### Requirements
Completed program may include:

- Case-insensitive letter input
- Dynamic word list loading from a file
- ASCII hangman drawing for each wrong guess
- Replay option after game over
